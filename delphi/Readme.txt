CODBAR
======
Este exemplo mostra como utilizar o codbar.exe de dentro de
uma aplicação Delphi. 

Como programa codbar foi desenvolvido em DOS,
qualquer aplicação em DOS poderá obter os mesmos resultados.

Esta versão está disponível também para qualquer plataforma que
tenha compilador 'C'.
